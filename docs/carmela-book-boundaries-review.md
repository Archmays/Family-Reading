# Carmela Season 1 PDF Book Boundaries Review

## Review Basis

The PDF has 320 pages. It appears to be image-based, so book boundaries were reviewed visually rather than by extracted PDF text.

For each book, at least these pages were visually sampled:

- Cover page
- First content page after the cover
- Last page before the next book boundary

These boundaries are temporary but stable enough for the next approved phase: PDF image extraction and white-border cropping.
